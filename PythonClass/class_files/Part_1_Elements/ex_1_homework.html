Install an editor!

For class purposes I recommend Sublime Text.

  1. Download and install Sublime Text
  2. Drag the ClassFiles folder to the Sublime Text app.
  3. Sublime Text will open a navigator for the folder.
  4. Select one of the scripts in Part 1 and use Command-B to run it.
  5. If you get a menu for "Python / Python Syntax check", choose "Python"!
